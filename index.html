1. Data Collection and Usage

What data is collected

We collect personal information such as email addresses (via Firebase Authentication), usage data (e.g., app activity), and technical information (e.g., device model and OS version).

How data is collected

Data is collected through user input, Firebase SDKs, and third-party APIs including ChatGPT, Claude, Gemini, DeepSeek, and Mathpix.

Why data is collected

We collect data to authenticate users, enhance app functionality, provide AI features, and personalize user experiences.

Data sharing

Data may be shared with Firebase, RevenueCat, and AI service providers (OpenAI, Anthropic, Google, Mathpix) solely to provide core app features. We do not sell user data.

Data retention and deletion

Data is stored in Firebase Firestore and retained until you request deletion. You can request this by contacting our support team.

Data security

We use Firebase Security Rules, authentication layers, and encrypted transmission (HTTPS) to protect your data.
